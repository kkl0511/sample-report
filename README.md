## 배포 방법 (GitHub Pages)

1. 이 저장소를 GitHub에 **Public**으로 올립니다.
2. 저장소 → **Settings → Pages → Source: `main` branch / root** → Save.
3. 약 1분 후 `https://<사용자명>.github.io/<저장소명>/` 에서 접속 가능합니다.
