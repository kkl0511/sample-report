# BBL v30.35 — 점수 의미 재정의 (잠재력 발전 가능성 + 또래 비교)
**Build**: 2026-05-03 / **Patch**: v30.34 → v30.35 / **Type**: 점수 의미 톤 변경

---

## 변경 요약 (사용자 통찰 반영)

### 1) "도달도 95% = 거의 발현" 표현 완전 제거

**이전 잘못된 해석**: 도달도 95% → "잠재력 거의 발현, 다 왔다"
**올바른 해석**: 95% → "현재 수준, 잠재력 5%(+7.4 km/h) 추가 발전 여지"

→ 헤더 카드 라벨 변경:
- "체력 도달도 86%" → **"체력 현재 수준 127.3 km/h"** + "잠재력 147.9 km/h 까지 발전 여지"
- "메카닉 도달도 86%" → **"메카닉 현재 수준 128.5 km/h"** + "잠재력 149.6 km/h 까지 발전 여지"
- "종합 도달도 95%" → **"잠재 구속 (FM100) 150.8 km/h"** + "+7.4 km/h 성장 잠재력"

### 2) 잠재력은 동적 개념 (시간에 따라 더 커짐)

prodigyNote 핵심 메시지 추가:
> **중요 — 잠재력은 고정된 한계가 아닙니다**:
> 이 잠재 구속은 *현 시점의 신체·메카닉 조건* 기준 추정값입니다.
> 향후 **신장·근력 성장**, **트레이닝**, **메카닉 정밀화**에 따라
> **잠재 구속 자체가 더 높아질 수 있습니다**.

### 3) 카테고리 차트 = "또래 대비 본인 위치"

**이전**: "체력" + "/100" (코호트 percentile, 의미 모호)
**변경**: "고교 또래 대비 내 수준" + "/100 percentile" + "같은 고교 코호트 중앙값 = 50점, 상위권 = 70점+"

→ 학년 변경 (고교 → 대학 → 프로) 시 비교 코호트 자동 변경 (현재 `r.age` 기반)

### 4) 종합 평가 1단계 카드 — "내 현재 수준과 발전 여지"

`도달도(%)` 대신 km/h 직접 표시 + 잠재력은 향후 더 높아질 수 있음 명시.

## 정예준 케이스 — 메시지 개선

**이전 (v30.34)**:
> 도달도 95% — "잠재력의 95%를 이미 발현한 elite 수준입니다"

**현재 (v30.35)**:
> 측정 구속 143.4 km/h · 잠재 구속 150.8 km/h · **성장 여지 +7.4 km/h**
>
> **잠재력은 고정된 한계가 아닙니다**: 향후 신장·근력 성장 + 트레이닝 + 메카닉 정밀화에 따라 잠재 구속 자체가 더 높아질 수 있습니다.

→ 95% 같은 비율 표시는 제거. 절대 수치(km/h) + 발전 여지 + 잠재력 동적 성장 가능성 강조.
